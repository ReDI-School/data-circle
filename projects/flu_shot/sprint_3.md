# Sprint 3: Insights & Deployment
**Week 7–9**

### Overview
In this final sprint, teams will focus on deriving actionable insights from their models, creating an interactive dashboard to present their findings, and preparing their final presentation.

1. Model interpretation and insight generation  

Use models as tools to extract meaningful insights from the data. This may include:  
- Feature importance analysis (e.g., model-based importance, SHAP values)  
- Investigating non-linear relationships and interactions  
- Comparing insights across different models  
- Relating model findings back to your original hypothesis

2. Reflect on what the model and data cannot explain.

3. Dashboard development (optional) and communication  

Translate your findings into a clear and interactive data product.  
- Build an interactive dashboard (e.g., Streamlit)  
- Include key insights, visualizations, and  model outputs (optional) 
- Use maps where relevant  
- Focus on clarity, usability, and storytelling

### Deliverables
- Interactive Streamlit dashboard (Optional)
- GitHub repository with complete code and documentation
- Interpretation of results and key insights
- 5-minute final presentation
